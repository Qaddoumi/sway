## 🏔️ Installation 🏔️

- Run :

``` shell
bash <(curl -sL https://raw.githubusercontent.com/Qaddoumi/sway/main/install.sh)
```

- Or for only the config files :

``` shell
bash <(curl -sL https://raw.githubusercontent.com/Qaddoumi/sway/main/installconfig.sh)
```
